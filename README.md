# github-project-automation

> A GitHub App built with [Probot](https://github.com/probot/probot) that automates your GitHub Project by adding new issues and PRs

## Setup

You need Node >= 12.16

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
